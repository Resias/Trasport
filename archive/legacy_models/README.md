# Legacy Models

This folder contains source code kept for historical reference but excluded from the
Appl. Sci. 2026 main experiment workflow.

Archived items:

- `SCIE_Benchmark/STDAMHGN.py`
- `SCIE_Benchmark/hyper_graph_preprocessed.py`
- `train/train_st_damhgn.py`

These files were not part of the paper's main comparison table. They are retained so old
experiments can be inspected or restored, but new Appl. Sci. 2026 reproduction commands should use
`experiments/applsci_2026/`.
